## Run Locally

Clone the project

```bash
  git clone https://github.com/devshinthant/therealhotels.git
```

Go to the project directory

```bash
  cd therealhotels
```

Install dependencies

```bash
  pnpm install
```

Start the server

```bash
  pnpm start
```
